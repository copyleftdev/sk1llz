# Bjarne Stroustrup References

## Books

### Essential Reading

1. **"The C++ Programming Language" (4th Edition, 2013)**
   - The definitive reference by the creator
   - Covers C++11 comprehensively
   - Philosophy interwoven with technical detail

2. **"A Tour of C++" (3rd Edition, 2022)**
   - Concise overview of modern C++
   - Best starting point for experienced programmers
   - Covers through C++20

3. **"The Design and Evolution of C++" (1994)**
   - Why C++ is the way it is
   - Essential for understanding design decisions
   - Historical context for every major feature

### C++ Core Guidelines

- **[C++ Core Guidelines](https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines)**
  - Co-authored with Herb Sutter
  - Living document of best practices
  - Enforceable via static analysis

## Talks

### Foundational

1. **"Writing Good C++14" (CppCon 2015)**
   - Modern C++ philosophy in practice
   - RAII, type safety, static analysis

2. **"The Essence of C++" (University of Edinburgh, 2014)**
   - Core ideas distilled
   - Why C++ makes the choices it does

3. **"Learning and Teaching Modern C++" (CppCon 2017)**
   - How to approach the language
   - What to teach first

### Recent Evolution

4. **"C++20: Reaching for the Aims of C++" (CppCon 2021)**
   - How C++20 fulfills original goals
   - Concepts, coroutines, modules

5. **"Delivering Safe C++" (CppNorth 2023)**
   - Modern safety approach
   - Profiles proposal

## Key Concepts to Study

From Stroustrup's work, prioritize understanding:

1. **RAII** - Resource Acquisition Is Initialization
2. **Zero-overhead principle** - Abstraction without cost
3. **Type safety** - Compile-time error detection
4. **Value semantics** - Objects as values, not references
5. **Concepts** - Constrained templates (C++20)
