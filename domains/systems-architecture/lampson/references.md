# Butler Lampson: References

## Essential Papers

### System Design

- **"Hints for Computer System Design"** (1983)
  - The foundational paper on system design wisdom
  - Required reading for every systems engineer
  - [ACM SIGOPS](https://dl.acm.org/doi/10.1145/800217.806614)
  - [Updated version (2011)](https://www.microsoft.com/en-us/research/publication/hints-and-principles-for-computer-system-design/)

### Networking and Distributed Systems

- **"End-to-End Arguments in System Design"** (1984)
  - Saltzer, Reed, Clark (Lampson influence)
  - The end-to-end argument explained
  - [ACM TOCS](https://dl.acm.org/doi/10.1145/357401.357402)

- **"The Byzantine Generals Problem"** (1982)
  - Lamport, Shostak, Pease (Lampson contributed)
  - Foundational BFT work

### Security

- **"Protection"** (1974)
  - Foundational work on capability-based security
  - [ACM SIGOPS](https://dl.acm.org/doi/10.1145/775265.775268)

- **"Authentication in Distributed Systems"** (1992)
  - With Abadi, Burrows, Wobber
  - Theory and practice of authentication
  - [ACM TOCS](https://dl.acm.org/doi/10.1145/138873.138874)

### Programming Languages and Compilers

- **"A Theory of Type Polymorphism in Programming"** (context)
  - Milner (Lampson contributed to type theory discourse)

### The Alto and Personal Computing

- **"Personal Distributed Computing: The Alto and Ethernet Software"** (1979)
  - Describes the Xerox Alto system
  - [ACM Conference](https://dl.acm.org/doi/10.1145/1499949.1500041)

## Historical Context

### Xerox PARC (1970s)

- Alto personal computer (first GUI)
- Ethernet (co-invented with Metcalfe)
- Laser printing
- WYSIWYG editing

### DEC SRC (1980s–1990s)

- Distributed systems research
- Modula-3 language
- Security foundations

### Microsoft Research (1995–present)

- Updated "Hints" paper
- Security protocols
- Systems architecture guidance
