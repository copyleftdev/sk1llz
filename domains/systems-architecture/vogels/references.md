## Historical Context

### Amazon's Evolution

- **1994–2000**: Monolithic application
- **2001–2005**: Service-oriented architecture
- **2006**: AWS launches (EC2, S3)
- **2007**: Dynamo paper published
- **2010s**: Microservices, serverless
- **2020s**: Multi-region, edge computing

Vogels has guided Amazon's architecture through each transition.
