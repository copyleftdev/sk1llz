# Jeff Dean: References

## Essential Papers

### MapReduce
- **"MapReduce: Simplified Data Processing on Large Clusters"** (2004)
  - Dean & Ghemawat
  - The paper that democratized large-scale data processing
  - [Google Research](https://research.google/pubs/pub62/)

### BigTable
- **"Bigtable: A Distributed Storage System for Structured Data"** (2006)
  - Chang, Dean, Ghemawat, et al.
  - Foundation for HBase, Cassandra, and others
  - [Google Research](https://research.google/pubs/pub27898/)

### Spanner
- **"Spanner: Google's Globally-Distributed Database"** (2012)
  - Corbett, Dean, et al.
  - First globally-consistent distributed database
  - [Google Research](https://research.google/pubs/pub39966/)

- **"Spanner, TrueTime and the CAP Theorem"** (2017)
  - Brewer (with Dean's input)
  - Clarifying how Spanner relates to CAP
  - [Google Research](https://research.google/pubs/pub45855/)

### TensorFlow
- **"TensorFlow: A System for Large-Scale Machine Learning"** (2016)
  - Abadi, Barham, Chen, et al. (Dean as co-author)
  - ML infrastructure at scale
  - [OSDI 2016](https://www.usenix.org/conference/osdi16/technical-sessions/presentation/abadi)

### Infrastructure
- **"The Google File System"** (2003)
  - Ghemawat, Gobioff, Leung (Dean-adjacent)
  - Foundation for distributed storage
  - [Google Research](https://research.google/pubs/pub51/)
